# B - Roller Coaster

## 問題文

高橋君の仲間たちは N 人で遊園地に遊びにいきました。  
遊園地の一番人気のジェットコースターに乗るためには、身長が K cm以上必要です。  
高橋君の i 番目の仲間の身長は h_i cm です。  
高橋君の仲間たちのうち、一番人気のジェットコースターに乗ることができる人の数を求めてください。 

## 制約

- 1≤N≤10^5
- 1≤K≤500
- 1≤hi≤500
- 入力はすべて整数

## 入力

```bash
N K
h_1 h_2 … h_N
```

## 出力

高橋君の仲間たちのうち、一番人気のジェットコースターに乗ることのできる人の数を出力してください。
